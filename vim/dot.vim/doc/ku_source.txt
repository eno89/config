INTRODUCTION					*ku-source-introduction*

"source" is a source for ku, a Vim plugin (|ku.txt|).  This source gathers all
available sources.




==============================================================================
ACTION TABLE					*ku-source-action-table*

default						*ku-source-aciton-default*
			Same as "switch".

switch						*ku-source-aciton-switch*
			Switch the current source to the selected item.
			If the selected item is not completed one, it is an
			error and nothing will be happened.
